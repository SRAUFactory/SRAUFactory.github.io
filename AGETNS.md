# SRAUFactory.github.io AI Agent Instructions (AGETNS.md)

このファイルは、本プロジェクトを操作するAIエージェント向けの指示書です。本プロジェクトの開発、修正、または調査を行う際は、以下の概要および制約事項に従ってください。

## プロジェクト概要

個人フリーランスエンジニアである SRAU Factory の公式ホームページとなる静的Webサイトプロジェクトです。

### 技術スタック（コードから確認できる事実に基づく情報）
- **フロントエンド基本技術**: HTML5, Vanilla CSS, Vanilla JavaScript
- **スタイル（CSS）**: `css/index.css` に定義されたカスタムスタイルを使用
- **スクリプト（JS）**: `js/main.js` にてモバイルメニューの開閉トグルや、アンカーリンクへのスムーズスクロール処理を実装
- **アナリティクス**: Simple Analytics（`https://scripts.simpleanalyticscdn.com/latest.js` の読み込み）
- **静的サイト生成/ビルド**: リポジトリ内にビルドツール（Vite, Webpack, Babel等）の構成ファイルが存在しないため、ソースコードが直接Webサーバーから配信される静的サイトであると推測されます。

### ディレクトリ構成
- `index.html`: 公式サイトのメインページ
- `css/index.css`: スタイル定義ファイル
- `js/main.js`: インタラクティブ処理用スクリプトファイル
- `llms.txt`: LLM/AIエージェント向けにスキルの要約や経歴をまとめたテキストファイル
- `README.md`: プロジェクトの基本的な案内用マークダウンファイル
- `_config.yml`: GitHub Pages（Jekyll）のビルド設定ファイル（Markdownファイルの除外設定など）

## AIエージェントへの制約事項

本プロジェクトを扱うAIエージェントは、以下の制約事項を厳守しなければなりません。

### 1. 日本語による出力
すべての出力（返答、コミットメッセージ、ドキュメントなど）は、原則として日本語で行ってください。

### 2. 事実と推測の明確な分離および非断定的な記述
- **事実（確証が得られている情報）**と**推測（仮説やソースコードから読み取れるが明示的な裏付けがない情報）**を明確に区別して記載してください。
- 根拠のない内容について断定表現（〜である、〜だ）を使用せず、推測である場合はその旨（〜と推測される、〜と考えられる）を明記してください。
